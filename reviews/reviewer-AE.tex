\comment{
In the introduction, several methods for analyzing shot charts are summarized, including several based on intensity surfaces. The authors cite the requirement of a discretized grid for these methods as the reason for pursuing a density-based analysis, but this is not convincing. All functional data analyses must, in the end, be performed on a discrete grid, and it is not obvious that the estimated (log) intensity surfaces could not also be used as functional data objects. Another key difference between bivariate densities and intensity surfaces is that the latter contain information about shot volume, an aspect completely absent in the density representation. Please provide a more thorough justification for using densities as data objects as opposed to existing approaches.
}

\reply{
	{\color{purple}TO DO = ADD ABOUT BINS AS DISCUSSED IN MEETING, INTEGRATE INT DO RESULTS AND DISCUSSION.}
	\begin{itemize}
		\item \textbf{Intensity surfaces}: mean number of shots expected per unit area. These are \emph{not} normalized. Players who play more games, or just shoot more in general can have different intensities despite having same underlying patterns.
		\item \textbf{Density surfaces}: These are normalized (densities integrate to $1$), hence more suitable for comparing shot profiles, especially across longer time periods. The only drawback is that they can lack information shot volumes.
	\end{itemize}

	So our justification:
	\begin{itemize}
		\item We use densities because they are normalized so to make fair comparisons of \emph{patterns/ densities} that are not affecte by, e.g., number of games or minutes played.
		\item To DO: Present a histogram of total number of shots per player to show that these can inflate/ mislead an intensity based approach.
		\item Acknowledge that an overall shot volume metric, normalized where appropriate, e.g., total shots per game, could lead to an improved characterization in future work.
	\end{itemize}

\comment{
While it is true that early work in functional data analysis did include applications of FPCA (or other standard FDA methods) directly to densities \parencite{kneipInferenceDensityFamilies2001,neriniClassifyingDensitiesUsing2007, ramsayFunctionalDataAnalysis2005}, these methods have been superseded in the last decade by non-linear dimension reduction methods due to the fact that density properties are not retained by linear decompositions \parencite{bigotGeodesicPCAWasserstein2017, delicadoDimensionalityReductionWhen2011, petersenFunctionalDataAnalysis2016}. Many if not all of these newer methods can only be applied to univariate densities; however, the authors should include a discussion about the potential pitfalls of FPCA when applied to functional data that, by nature, satisfy nonlinear constraints (positivity and integration to 1). If possible, some attempt to mitigate these issues for the bivariate shot chart densities should be made.
}

\reply{
	{\color{purple}To add to paper, in discussion as a limittation:}
	\begin{quote}
		\emph{We have applied FPCA, a conventional linear dimension reduction technique for standard functional data, i.e., curves assumed to belong to the space of $\mathcal{L}_2$ of square-integrable functions. However, densities belong to a constrained space -- they are non-negative and integrate to $1$.
			As a result, reconstructions given by standard FPCA, are not guaranteed to satisfy density constraints.
			As such, alternative non-linear approaches have been proposed for modeling univariate densities, e.g., transformations that map the densities to an unconstrained space before doing FPCA \parencite{petersenFunctionalDataAnalysis2016}.
			However, we have chosen to apply standard FPCA, acknowledging its limitations, because:
			\begin{enumerate}
				\item Conventional FPCA is tractable, it is straightforward to describe to a general data-science audience. We can interpret the individual FPCs as a shift of the mean density \parencite{kneipInferenceDensityFamilies2001}. Furthermore, standard computational FPCA methods can be employed to deal with the high-dimensionality of the two-dimensional surfaces.
				\item Although specialized geometrically-aware density approaches can be can give better performance, standard FPCA has been shown to provide a reasonable approximation in many cases \parencite{petersenFunctionalDataAnalysis2016} \textbf{(CITE DAI and Muller)}.
				\item If FPCA approximations of individual densities take negative values, they can be projected back into the density space, by retaining only the positive part and re-normalizing. \parencite{petersenFunctionalDataAnalysis2016} \textbf{(CITE HALL)}
			\end{enumerate}
		}
		Therefore, we decided to use the standard FPCA approximation rather than develop a new method by extending existing univariate density approaches to the bivariate case.
	\end{quote}
}
